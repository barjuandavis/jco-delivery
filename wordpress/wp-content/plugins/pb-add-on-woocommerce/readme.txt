== Description ==

Profile Builder - WooCommerce Sync Add-on

Syncs Profile Builder with WooCommerce, allowing you to manage the Shipping and Billing fields from WoCommerce with Profile Builder.

NOTE:

This plugin requires and is compatible with Profile Builder version 2.0 or higher

== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to Manage Fields tab under Profile Builder menu and you'll have access to WooCommerce Shipping and Billing fields.
